**Dynamic port Forwarding**
Dynamic port forwardiing creats  a socks proxy using ssh.It allows application (Browser,Curl) to route traffic dynamically to any destinantion through an encrypted SSH tunnel


**Lab setup**

1) VM1:- local server
2) VM2:- Web server
3) Nginx services on port 80
4) SSH Services on port 22
5) Blocked port 80 on web server



**Prerequisites**
SSH access to remote server
. Web Service runnig on remote Server
.SSh service runnig
Port 80 Blocked on web server
